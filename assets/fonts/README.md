# Plus Jakarta Sans Font

Please download the Plus Jakarta Sans font from Google Fonts and place the following files here:

1. PlusJakartaSans-Light.ttf (300)
2. PlusJakartaSans-Regular.ttf (400)
3. PlusJakartaSans-Medium.ttf (500)
4. PlusJakartaSans-SemiBold.ttf (600)
5. PlusJakartaSans-Bold.ttf (700)

Download link: https://fonts.google.com/specimen/Plus+Jakarta+Sans

**For now, the app will use system fonts until you add these files.**
